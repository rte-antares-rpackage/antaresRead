# View the content of an antares output

This function displays each element of an `antaresData` object in a
spreadsheet-like viewer.

## Usage

``` r
viewAntares(x, ...)
```

## Arguments

- x:

  An object of class `antaresData`, generated by the function
  [`readAntares`](readAntares.md).

- ...:

  Currently unused

## Value

Invisible NULL.

## Examples

``` r
if (FALSE) { # \dontrun{
setSimulationPath()

areas <-readAntares()
viewAntares(areas)

output <- studyAntares(areas="all", links = "all", clusters = "all")
viewAntares(output) # Opens three data viewers for each element of output
} # }
```
