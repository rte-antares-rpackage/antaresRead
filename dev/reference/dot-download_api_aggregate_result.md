# Retrieve information on a file's state of preparation and retrieve download file.

Retrieve information on a file's state of preparation and retrieve
download file.

## Usage

``` r
.download_api_aggregate_result(download_id, opts)
```

## Arguments

- download_id:

  the id of the download.

- opts:

  List of simulation parameters returned by the function
  [`setSimulationPath()`](setSimulationPath.md)

## Value

An updated list containing various information about the simulation.

a data.table
