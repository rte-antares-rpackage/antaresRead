# Write digest file

Write digest file

## Usage

``` r
writeDigest(digest, opts = simOptions())
```

## Arguments

- digest:

  list of 5 elements similar to what is returned by
  [`readDigestFile`](readDigestFile.md)

- opts:

  simulation options

## Value

updated digest list of 5 tables (begin, areas, middle, links lin., links
quad.)
