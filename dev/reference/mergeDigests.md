# Merge two digests

Merge two digests

## Usage

``` r
mergeDigests(digest_new, digest_ori)
```

## Arguments

- digest_new:

  new digest with missing lines

- digest_ori:

  original digest with all lines

## Value

updated digest list of 5 tables (begin, areas, middle, links lin., links
quad.)
